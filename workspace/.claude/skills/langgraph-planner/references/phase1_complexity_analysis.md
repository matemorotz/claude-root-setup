## Detailed Process

**Goal:** Understand what the graph needs to accomplish

I will ask you:
1. What is the primary purpose of this graph?
2. What are the main tasks it will handle?
3. Is this a single-agent workflow or multi-agent orchestration?
4. What external systems will it integrate with? (APIs, databases, MCP tools)
5. Estimated complexity: simple (3-5 nodes) or complex (7+ nodes)?

**Output:** Graph purpose statement + complexity assessment + architecture recommendation
